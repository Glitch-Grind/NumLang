# NumLang IDE

NumLang is a **mini programming language** with its own syntax. This repository contains a **canvas-based IDE** with syntax highlighting and a “Run NumLang” button.

---

## Features

- Canvas-based code editor  
- Syntax highlighting for keywords, numbers, strings, and comments  
- Left-to-right typing and single-line Enter spacing  
- Run NumLang scripts directly from the browser  
- Dark theme, modern look, and stable caret
